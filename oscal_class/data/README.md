# OSCAL Minimun Artifacts

The files in this folder are intended for creating new OSCAL artifacts.

The corrisponding XML file for each model contains the minimum necessary syntax for the file to be valid while including as little content as possible. 

When a user indicates they want to create a new OSCAL artifact, the OSCAL class uses these files to initiate the content.

In a custom application, these files can be pre-populated as appropriate, such as with organizational details. Anything that would be useful to the user when creating a new artifct is allowed, provided the additions are valid OSCAL syntax.
