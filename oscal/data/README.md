# OSCAL Minimun Artifacts

These files in this folder are used by the `OSCAL.new()` method when creating 
new OSCAL artiacts. 

Each file contains the minimum necessary syntax for the file to be valid while including as little content as possible. 

In a custom application, these files can be pre-populated as appropriate, such as with organizational details. Anything that would be useful to the user when creating a new artifct is allowed, provided the additions are valid OSCAL syntax.
